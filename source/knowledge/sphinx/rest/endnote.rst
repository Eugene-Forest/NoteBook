脚注/尾注
===============

使用 [#name]_ 标记脚注位置，并将脚注正文添加到文档底部的“footnotes”标题后面，

.. code-block:: rest

   脚注参考 [5]_

   ..  [5] 这是对应的注记信息

   自动脚注

   脚注参考 [#]_，这是第二个脚注 [#]_。

   ..  [#] 这是第一个注记的信息
   ..  [#] 这是第二个注记的信息

.. note:: 
   运行效果如下方所示。

----
