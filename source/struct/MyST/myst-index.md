# MyST 语法学习

MyST 是一种丰富且可扩展的 Markdown 风格，用于技术文档和发布。



```{note} 可以同时使用 MyST 和 reStructuredText
激活 MyST 解析器将简单地启用 MyST解析 Markdown 文件，Sphinx 附带的 rST 解析器仍将以相同的方式工作。以 结尾的文件.md将被解析为 MyST，以 结尾的文件.rst将被解析为 reStructuredText。
```
