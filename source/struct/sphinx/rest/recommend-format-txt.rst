标题之间的分隔
===================


在此约定，无论标题形式如何，统计标题之间的分隔符长度相同，而不同级别之间的标题之间的分隔符的长度比例从大到小为 75% 、 50% 、 30% 、15% 。（笔者认为一篇文章中除了主标题外，最好只出现最多三级标题，再多后边不利于编写和观看。）

.. code-block:: rest

    .. raw:: html

        <hr width='75%'>

    .. raw:: html

        <hr width='50%'>

    .. raw:: html

        <hr width='30%'>

    .. raw:: html

        <hr width='15%'>
